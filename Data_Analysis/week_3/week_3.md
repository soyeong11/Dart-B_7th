# 1️⃣ 개념 정리 

## 01. 불필요한 데이터 삭제하기

### 📍데이터 삭제

#### 📌열 삭제

**1. drop() 메서드**

데이터프레임의 행 or 열 삭제
```python
데이터프레임.drop('삭제하려는 열 이름', axis=1)
# axis=1은 열 삭제
# 삭제 열 여러 개면 ['열 이름', '열 이름']의 리스트 형식 사용
```
inplace 매개변수
```python
데이터프레임.drop('삭제하려는 열 이름', axis=1, inplace=True)
# 현재 선택한 데이터프레임 바로 수정 가능
```

**2. dropna() 메서드**

NaN이 하나 이상 포함된 행 or 열 삭제
```python
데이터프레임.dropna(axis=1) # 또는 axis=0
```

how 매개변수
```python
데이터프레임.dropna(axis=1, how='all')
# how='all'은 모든 값이 NaN인 행 or 열 삭제
```

#### 📌행 삭제

**1. drop() 메서드**

axis=0 지정 or 기본값이 0이기 때문에 생략 가능
```python
데이터프레임.drop([0,1])
# 인덱스 0부터 1까지 선택해 삭제
```

**2. []연산자와 슬라이싱**

- ex: 인덱스 0, 1 행 제외 모든 행 선택 = [2:]
- ex: 처음 두 행만 선택 (0, 1만 선택) = [0:2]


**3. []연산자와 불리언 배열**

```python
불리언_배열_이름 = 데이터프레임['열이름'] == '뽑고 싶은 열 이름'
새로_만들_데이터프레임 = 데이터프레임[불리언_배열_이름]
```
'열 이름'이 '뽑고 싶은 열 이름'인 행만 뽑힘

- ex:
```python
ns_book2 = ns_book[ns_book['대출건수'] > 1000]
# 대출건수가 100건 이상인 열만 뽑기
```

### 📍중복된 행 찾기

**1. duplicated() 메서드**

데이터프레임에 있는 모든 열을 기준으로 중복된 행 찾기

subset 매개변수에 기준열 나열

```python
데이터프레임.duplicated(subset=['열 이름', '열 이름', '열 이름'])
# keep=False 추가하면 중복된 행을 모두 True로 표시한 불리언 배열을 반환
```

### 📍그룹별로 모으기

**1. groupby() 메셔드**

- groupby() 메서드 + by 매개변수 (기준이 되는 열 지정)
- 기본적으로 by 매개변수에 지정된 열에 NaN이 포함되어 있으면 해당 행 삭제
    - 원하지 않으면 dropna 매개변수를 False로 지정

- ex:
```python
데이터프레임.groupby(by=['열 이름', '열 이름', '..'], dropna=False)
```

### 📍원본 데이터 업데이트

**1. 순서**

(1) duplicated()로 중보된 행을 True로 표시
```python
데이터프레임.duplicated(subset=['열 이름', '열 이름', '..'])
```
(2) 1번에서 구한 불리언 배열을 반전시켜 중복되지 않은 행을 True로 표시
(3) 2번에서 구한 불리언 배열로 원본 배열에서 고유한 행만 선택

**2. ~ 연산자**

순서 2번에서 불리언 배열을 반전시킬 때 사용되는 판다스 연산자

**3. copy() 메서드**

순서 3번에서 고유한 배열을 선택해서 복사

**4. set_index() 메서드**

지정한 열 인덱스로 설정 <br>
**inplace 매개변수**를 True로 지정: 새로운 데이터프레임 반환하지 않고 원본 수정

**5. update() 메서드**

다른 데이터프레임을 사용해 원본 데이터프레임 값 업데이트 <br>
**reset_index()**로 데이터프레임 인덱스 재설정 가능


### 📍한바닥 정리

1) 데이터 정제란
- 수집된 데이터에서 잘못된 부분을 고치거나 제거하여 필요한 데이터를 준비하는 과정
- 잘못된 정제는 분석 결과를 왜곡과 잘못도니 의사 결정 초래 가능

2) 데이터 랭글링 (데이터 먼징)
- 데이터를 정제하는 과정과 데이터 분석 & 머신러닝에 적합한 형태로 데이터를 변환하는 과정

3) 원소별 비교
- 판다스의 데이터프레임과 인덱스를 하나의 값과 비교 → 데이터프레임과 인덱스에 있는 모든 원소와 비교
- 비교한 결과는 True or False의 **불리언 배열**로 반환

4) 넘파이
- 대표적인 파이썬 다차원 배열
- 한 종류의 데이터만 저장 가능
- 다른 과학 패키지와의 호환성 good

5) 핵심 함수와 메서드

| 함수/메서드 | 기능 |
| --- | --- |
| DataFrame.drop() | 데이터프레임의 행 or 열 삭제 |
| DataFrame.dropna() | 누락된 값이 포함된 행 or 열 삭제 |
| DataFrame.duplicated() | 중복된 행을 찾아 불리언 값으로 표시한 배열 반환 |
| DataFrame.groupby() | 데이터프레임의 행을 그룹으로 모음 |
| DataFrame.sum() | 행 or 열을 기준으로 합계 계산 |
| DataFrame.set_index() | 지정한 열을 인덱스로 설정 |
| DataFrame.reset_index() | 데이터프레임의 인덱스 재설정 |
| DataFrame.update() | 다른 데이터프레임을 사용해 원본 데이터프레임의 값 업데이트. 다른 데이터프레임에 있는 NaN은 업데이트에서 제외 |
| DataFrame.equals() | 다른 데이터프레임과 동일한 원소를 가졌는지 비교. 두 데이터프레임이 동일하면 True, 그렇지 않으면 False 반환 |



## 02. 잘못된 데이터 수정하기

### 📍누락된 값 처리

**1. isna() 메서드**

누락된 값 개수 확인

isna() + sum() → 불리언 배열의 True 개수로 비어 있는 행 개수 반환

**2. None과 np.nan**

정수를 저장하는 열에 None 입력 → 누락된 값으로 인식

**3. astype() 메서드**

데이터 타입 지정 <br>
```python
데이터프레임.astype({'열 이름':'데이터 타입', '..':'..'})
```

**4. loc, fillna() 메서드**

- loc[] 메서드는 불리언 배열 필요
- isna()로 불리언 만들 수 있음
- fillna()에 원하는 값을 넣으면 NaN 대체 가능
- ex:
```python
데이터프레임.fillna({'열 이름'}: '바꿀 값').isna().sum()
```

**5. replace() 메서드**

- 바꾸는 값 하나
```python
replace(원래 값, 새로운 값)
```
- 바꾸는 값 여러 개
```python
replace([원래 값1, 원래 값2], [새로운 값1, 새로운 값2])
```

- 열마다 다른 값
```python
replace({열 이름: 원래 값}, 새로운 값)
```

### 📍정규 표현식 (정규식)

문자열 패턴을 찾아서 대체하기 위한 규칙 모음

**1. 숫자 찾기: /d**

- 숫자 기호: /d
- 연도 기호: /d/d/d/d
- 그룹으로 묶을 때: /d/d(/d/d)
- regex=True: 정규 표현식을 사용한다는 의미
- r 문자: 정규 표현식을 다른 문자열과 구분하기 위한 접두사
- ex:
```python
replace({'열 이름': {r'/d/d(/d/d)': r'/1'}}, regex=True)[100:102]
```
- 정규 표현식이 반복되면 중괄호로 개수 지정 가능
- ex:
```python
replace({'열 이름': {r'/d{2}(/d{2})': r'/1'}}, regex=True)[100:102]
```

**2. 문자 찾기: 마침표(.)**

어떤 문자에도 대응하는 정규 표현식 문자
- *문자: 0개 이상 반복된다는 표시
- \(역슬래시): 일반 문자라고 인식


### 📍잘못된 값 바꾸기

**2. gt() 메서드**

전달된 값보다 큰 값을 찾는 메서드


### 📍누락된 정보 채우기
