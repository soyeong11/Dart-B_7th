# 1️⃣ 개념 정리 

## 01. 데이터 분석이란

### (01) 데이터 분석과 데이터 과학
 '데이터 분석'이란 유용한 정보를 발견하고 결론을 유추하거나, 의사 결정을 돕기 위해 데이터를 조사, 정제, 변환, 모델링하는 과정. '데이터 과학'은 통계학 데이터 분석, 머신러닝, 데이터 마이닝 등을 아우르는 큰 개념.

 '데이터 과학'은 데이터 세계와 비즈니스 세계를 잇는 다리. 데이터 과학에는 프로그래밍, 통계학, 시각화와 더불어 비즈니스 감각일 필요함. 

 '데이터 분석'은 올바른 의사 결정을 돕기 위한 통찰 제공. '데이터 과학'은 문제 해결을 위한 최선의 솔루션 제작.

 통계학과 관련이 많고, 통계적 관점에서 보면 '기술통계', '탐색적 데이터 분석', '가설검정'의 세 가지로 데이터 분석을 나눌 수 있음.

### (02) 데이터 분석가
 **프로그래밍 기술을 갖추고 있고 통계학을 이해하면서 정보를 시각적으로 잘 표현할 수 있는 사람**
 프로그래밍, 수학, 통계, 도메인 지식이 필요함.
 작업 과정은 데이터 수집, 데이터 처리, 데이터 정제, 데이터 분석, 모델링으로 이루어짐.

### (03) 데이터 분석 도구
 - Python: 문법이 단순해서 쉽게 배울 수 있는 언어.
 - R: 통계 계산을 위해 개발된 언어. 데이터 수집, 전처리 등이 까다롭지만, 통계 관련 패키지는 Python보다 다양함.
 - 구글 코랩
 - 파이썬 필수 패키지
    - numpy
    - pandas
    - matplotlib
    - scipy
    - scikit-learn

## 02. 구글 코랩과 주피터 노트북

### (01) 구글코랩
 웹 브라우저에서 무료로 파이썬 프로그램을 테스트하고 저장할 수 있는 서비스. 작성한 코랩 파일은 노트북 혹은 코랩 메모장이라고 부름.
 
 코랩에서 실행할 수 있는 최소 단위는 셀로, 셀 안에 있는 내용은 한 번에 실행되고 그 결과가 노트북에 나타남.
 - 텍스트 셀: HTML과 마크다운을 혼용해서 사용 가능
 - 코드 셀: 파이썬 코드를 입력, 실행 가능

### (02) 코랩 VS 주피터 노트북
| 특징 | 구글 코랩 | 주피터 노트북 |
|------|----------|-------------|
| 설치 | 불필요 | 내 컴퓨터에 설치 |
| 실행 환경 | 인터넷이 연결된 웹 브라우저 | 오프라인 상태의 웹 브라우저 |
| 파일 저장 | 구글 드라이브에 자동 저장 | 내 컴퓨터 하드 디스크에 저장 |
| 패키지 | 사전 설치되어 제공 | 패키지 관리자로 직접 설치 |


## 03. 이 도서가 얼마나 인기가 좋을까요?
 "혼공출판사 도서의 판매 결과를 예측해보자."

### (01) 도서 데이터 찾기
 - 공개 데이터 세트 찾아보기
    - 공공데이터 포털에서 도서 판매 데이터 검색
         - 판매 데이터는 없지만, 도서관 데이터 존재 확인
 - 도서관 정보나루 사이트에서 남산 도서관 데이터 확인
    - CSV 파일 다운로드 (CSV: 콤마로 구분된 텍스트 파일)

### (02) 코랩에서 데이터 확인하기
 - gdown 패키지 사용 (pandas의 read_execl(), read_csv()로 파일 불러올수도 있음)
- 넘퓨터 파일 업로드: 왼쪽의 파일 창을 열고 CSV 파일을 드래그하여 넣는다.

### (03) 파이썬으로 CSV 파일 출력하기
- open()함수 사용
- with문으로 파일을 연 다음 readline() 매서드로 한 줄 출력
- chardet.detect(): chardet 패키지의 함수로, 문자 인코딩 방식을 알아낼 수 있음. 코랩에 이미 설치된 함수.

### (04) 데이터프레임 다루기: 판다스
 판다스는 CSV 파일을 읽어 데이터프레임이라는 표 형식 데이터(행과 열로 구성된 데이터 구조)로 저장.
 | 데이터프레임 | 시리즈 |
 |---------------|---------------|
 |행과 열로 구성. CSV와 엑셀 파일로 데이터프레임 제작 | 1차원 배열과 흡사한 판다스 객체. 한 종류의 데이터만 포함 |

 - read_csv(): CSV 파일을 데이터프레임으로 읽기
 - head(): 데이터프레임의 처음 다섯 개 행 확인
 - 첫 번째 열은 데이터프레임의 인덱스. 행마다 0부터 시작하는 인덱스 번호 자동 부여. 첫 행이 열 이름이 아니면 read_csv()함수를 호출할 때 header 매개젼수를 None으로 지정하고, names 매개변수에 열 이름 리스트 따로 전달.
 - to_csv(): 판다스의 데이터프레임을 CSV로 저장할 때 쓰는 매서드. UTF-8 형식으로 저장되기 때문에 encoding 매개변수 사용 불필요.

## 문제 1.

> **🧚Q. 데이터 분석의 개념을 넓은 의미와 좁은 의미로 나누어 설명하고, 두 개념의 차이점도 함께 정리해주세요.**


```
 좁은 의미의 데이터 분석은 수집된 데이터를 통계적으로 분석하는 방식 그 자체를 의미한다. 여기에는 세 가지의 분류가 있다.
 - 기술통계: 관측이나 실험을 통해 수집한 데이터를 정량화하거나 요약하는 기법
 - 탐색적 데이터 분석: 데이터를 시각적으로 표현하여 주요 특징을 찾고 분석하는 방법
- 가설검정은 데이터를 기반으로 특정 가정이 합당한지 평가하는 통계 방법

 넓은 의미의 데이터 분석은 좁은 의미의 데이터 분석을 포함하며, 데이터 수집, 처리, 정제, 그리고 모델링과 같은 데이터 분석의 전체적인 과정을 포함한다.

 좁은 의미의 데이터 분석은 데이터 분석 그 자체에 집중하지만, 넓은 의미의 데이터 분석은 데이터 수집부터 모델링까지의 모든 워크플로우를 포함한다.
```
